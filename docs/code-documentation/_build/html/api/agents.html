generate(resource_type, criteria=None, *, count=None, sort=None, interpreted_capability=None)[source]

Generate a relative FHIR search query URL.

Parameters:

  • resource_type (str) – FHIR resource (e.g. Patient, Observation).

  • criteria (dict[str, Any] | None) – Mapping of search parameter name → value. Values may be strings or {“comparator”: “ge”, “value”: “1990-01-01”} for dates.

  • count (int | None) – Optional _count paging parameter.

  • sort (str | None) – Optional _sort parameter.

  • interpreted_capability (dict[str, Any] | None) – When provided, flag criteria not declared on the target server’s CapabilityStatement.
