# Pitches & Leadership Summaries

This document contains different versions of the project pitch, tailored for various audiences.

---

## 30-Second Pitch (General Audience)

> "We built a demonstration showing how to apply **Software Factory principles** to agentic AI systems.  
> Instead of jumping straight into coding, we followed a structured approach — starting with clear planning and specifications, then building specialist agents with explicit feedback loops and human oversight.  
> Using Google ADK, we created a generalized FHIR query validator that doesn’t just validate queries, but also detects repeated errors, provides guidance, and escalates to humans when needed.  
> The goal was to show a repeatable, governable way of building agentic systems that are observable, safe, and aligned with real engineering discipline."

---

## 30-Second Pitch (Technical / Engineering Leadership)

> "We developed a demonstration of a **Software Factory approach** for building agentic systems using Google ADK.  
> Instead of ad-hoc agent development, we followed a structured process: creating detailed specifications first, then implementing specialist agents orchestrated through ADK Graph Workflows.  
> The system includes multiple explicit feedback loops — including a meta-loop for pattern detection that triggers either a learner agent or human escalation.  
> We also built in configuration-driven multi-server support and traceability across the workflow.  
> The goal was to establish a repeatable, observable, and governable pattern for future agentic initiatives."

---

## Slide Version (Leadership Deck)

**Slide Title:**  
**Applying Software Factory Principles to Agentic Systems**

**Key Points:**

- **Structured Approach**: Used phased planning, spec-driven development, and ADK Graph Workflows instead of ad-hoc agent building.
- **Explicit Feedback Loops**: Designed multiple loops — including a meta-loop that detects repeated failures and triggers either automated guidance or human intervention.
- **Governed & Observable**: Built-in traceability, configuration-driven multi-server support, and clear human oversight points for responsible AI deployment.

## Key Themes for All Pitches

- **Planning First**: Human-centric, high-leverage planning before implementation.
- **Spec-Driven Development**: Clear specifications drive agent behavior.
- **Loop Engineering**: Explicit feedback loops (including meta-learning).
- **Human Oversight**: Responsible escalation paths.
- **Traceability & Observability**: Making agent decisions visible and auditable.
- **Reusability**: A repeatable pattern for future agentic initiatives.
